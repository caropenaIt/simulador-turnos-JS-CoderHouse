Proyecto JS: modo oscuro + sistema de turnos

El proyecto consiste en crear un modo oscuro al sitio web para lograr un diseño más elegante y accesible.

Además, tendrá un sistema para pedir turno en una peluquería, eligiendo entre 4 peluqueros según días y horarios disponibles.

Proyecto creado por Carolina Alejandra Pena Astigarraga para el curso JavaScript Flex de CoderHouse 2024.
